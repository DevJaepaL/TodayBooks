## 🧱 기능
+ 사용자가 선택한 카테고리에 따라, 매일 오늘의 책을 소개해주는 기능입니다.
+ 사용자는 원하는 책의 상세 정보를 볼 수 있습니다.
+ 책에 개인 메모 및 별점을 통해 입력할 수 있습니다.
+ 읽었어요 버튼을 통해 사용자가 지금까지 읽은 책들을 모아볼 수 있습니다.
